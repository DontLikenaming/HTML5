링크 관련 태그

개요

현재 페이지에서 다른 페이지나 다른 사이트로 연결되도록 만드는 요소

이러한 요소를 하이퍼링크라 하며 간단히 링크라고 함

html에서는 a 태그를 이용해서 이 것을 만들 수 있는데
href 속성으로 문서 내 특정 위치, 연결할 다른 페이지나 외부사이트를 지정할 수 있음

a 태그 사이에는 링크를 설명할 텍스트를 작성함

문서 내 특정 위치로 이동

링크를 책갈피 용도로 생성할 수도 있는데
id라는 속성을 부여하여 책갈피 이름을 지정한 후, href 속성에 '♯책갈피 이름'을 지정하면
문서 내 만들어둔 특정 위치로 이동할 수 있음

문서 맨 위로

href 속성에 '파일명♯책갈피 이름'을 지정하면 특정 문서의 책갈피로도 이동할 수 있음

어제 강의 내용 중 entity reference로 이동

다른 태그 안에 id를 넣을 수도 있음 ex)<h2 id="...">

링크로 메일 보내기

메일로 보낼 내용을 작성할 수 있음
