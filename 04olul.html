목록 관련 태그

개요

깔끔한 정보전달을 위해 여러 요소들을 일렬로 나열한 명단이나 목록을 의미

html에서는 목록을 표현하기 위해 순서없는 목록, 순서 있는 목록, 정의 목록 등을 지원함

순서없는 목록(unordered)

명단의 요소를 기호로 나타내는 목록의 형태

ul, li 태그 등을 이용해 표현

목록에 사용 가능한 기호는 circle, square, disc 등이 있음

내가 좋아하는 점심메뉴

  • 돈까스
  • 월남쌈
  • 비빔밥
  • 고기만두
  • 국밥
  • 순살후라이드치킨

순서있는 목록(ordered)

명단의 요소를 순서가 있는 기호로 나타내는 목록의 형태

ol, li 태그 등을 이용해 표현

목록에 사용 가능한 기호는 decimal, alpha, roman 등 여러 종류가 있음

한글도 되긴 하는데 되는 브라우저가 있고 안되는 브라우저가 있음
브라우저에 따라 hangul 또는 -moz-hangul로 작동할 수도 있음

내가 좋아하는 점심메뉴

  1. 돈까스
  2. 월남쌈
  1. 비빔밥
  2. 고기만두
  1. 국밥
  2. 순살후라이드치킨
해당 목록은 작성자의 실제 기호 순위와 아무 연관 없음

정의 목록(definition)

용어와 그 용어에 대한 해설을 모아놓은 목록

dl, dt, dd 태그 등을 이용해 표현

웹 표준 용어

html
웹 사이트를 구성하는 웹 문서 작성에 사용하는 마크업 언어
CSS
html로 작성된 문서를 시각적으로 꾸밀 때 사용
Javascript
html에 사용하는 객체기반 스크립트 프로그래밍 언어

목록 중첩(nested)하기

목록을 구성하는 요소들을 구조화(structured)된 정보형태로 나타낼 때 주로 사용됨

목록 속에 또 다른 목록을 포함시킴
(ul->ul, ul->ol 등..)

내가 좋아하는 점심메뉴
